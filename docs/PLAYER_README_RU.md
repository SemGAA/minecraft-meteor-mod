# Meteor Event Mod - быстрый старт

Короткая инструкция для игрока без чтения исходников.

## Что нужно

- Minecraft Forge 1.21
- Файл мода `tutorialmod-0.0.1.jar`

## Скачать

- [Скачать jar](https://github.com/SemGAA/minecraft-meteor-mod/releases/latest/download/tutorialmod-0.0.1.jar)
- [Открыть страницу релизов](https://github.com/SemGAA/minecraft-meteor-mod/releases/latest)

## Установка

1. Установите Minecraft Forge 1.21.
2. Скачайте `tutorialmod-0.0.1.jar`.
3. Переместите jar в папку `.minecraft/mods`.
4. Запустите игру через профиль Forge.

## Проверка в игре

- Создайте или откройте мир.
- Для быстрого теста используйте команду `/meteor start`.
- После запуска события в мире появится метеорит с визуальными эффектами и кратером.

## Если мод не появился

- Проверьте, что версия Forge соответствует Minecraft 1.21.
- Проверьте, что jar лежит именно в `.minecraft/mods`.
- Убедитесь, что в папке `mods` нет старой конфликтующей версии этого же мода.
